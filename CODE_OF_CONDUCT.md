CODE OF CONDUCT – SIRIUS LOCAL AI ALFA
This project follows a professional and respectful standard of behavior. The goal is to maintain a safe, friendly, and productive environment for all contributors.

1. Core Principles
respect toward all participants

no personal attacks

no insults, mockery, or humiliation

no harassment of any kind

constructive and technical communication

disagreements handled calmly and professionally

2. Acceptable Behavior
helping other contributors

clear and direct communication

respecting different opinions

focusing on solving problems, not creating conflict

following project rules and guidelines

3. Unacceptable Behavior
aggressive or hostile communication

sexist, racist, or discriminatory comments

personal attacks or humiliation

spam, trolling, or intentional disruption

sharing private information without consent

4. Responsibilities of Project Maintainers
Project maintainers have the right and responsibility to:

remove comments that violate this code

block users who repeatedly break the rules

maintain a safe and professional environment

5. Reporting Issues
If any inappropriate behavior or conflict occurs, it can be reported directly to the project maintainer through GitHub Issues or private message.

All reports will be handled confidentially.

6. Consequences of Violations
Violations of this Code of Conduct may result in:

a warning

removal of comments

temporary or permanent ban from the project

7. Acknowledgment
By contributing to this project, you agree to follow this Code of Conduct.
